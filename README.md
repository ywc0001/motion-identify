# 健身管家

#### 介绍
基于mediapipe、opencv、PyQt5构建的健身动作检测程序，检测范围包括仰卧、俯卧、深蹲、引体、哑铃右侧屈伸、哑铃两侧屈伸和哑铃向上推举动作。 


#### 软件架构
1. 控制台模块
2. 哑铃模块
3. 俯卧撑、仰卧起坐、深蹲、引体向上模块


#### 安装教程与使用环境

1.  Windows10及以上版本的PC
2. 基于3.10版本的python
3. IDE使用pycharm
4. 文件中安装opencv及mediapipe-python解释器
5. 文件中安装PyQt5及pandas解释器
